# Project Overview

WorldCup2026Hub is intended to become a central hub for 2026 World Cup-related information and tooling.

## Current Scope

The repository is in its initial setup phase. The first priority is to keep project intent, operational notes, and future technical decisions documented in a lightweight way.

## Potential Areas

- Tournament information and schedules
- Host city and venue notes
- Team, group, and match data
- Travel or fan-facing reference material
- Internal planning and implementation notes

## Documentation Principles

- Keep pages short and focused.
- Prefer clear links over duplicated content.
- Record decisions when they affect future implementation work.
