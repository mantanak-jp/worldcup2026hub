# Source Policy

WorldCup2026Hub treats external sources as references for personal research. The repository should store metadata and concise original notes, not copied content.

## Allowed Source Metadata

- Source name
- URL
- Language
- Source type, such as official, media, federation, or statistics
- Related match IDs
- Related team IDs
- Checked status
- Short original Japanese notes

## Not Allowed Without Explicit User Approval

- Copied article bodies
- Downloaded or redistributed external images
- Large quotations
- Automated crawling targets
- Terms-of-service-sensitive extraction

## Review Status

- `candidate`: source has been identified but not reviewed.
- `manual-review-needed`: source requires human review before use.
- `approved-reference`: source is approved for metadata and link use.
- `blocked`: source should not be used.

## Notes

Japanese notes should be concise and original. They should summarize why the source matters, not reproduce article text.
